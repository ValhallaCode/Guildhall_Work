Control:
Up Arrow - move Up
Down Arrow - move Down
Left Arrow - move Left
Right Arrow - move Right
Z - Sleep for 1 turn
P - View Stats Menu
Esc - Move back one menu / close App

Feature:
Door will loose back color when interacted with.
Blue numbers are a defense miss.
Light Blue numbers are an evade miss.

Symbols:
@ = Player
O = Orc
+ = Door

Known Bugs:
A* pathing will occasionally fail to genereate a path.
There is no Debug Step for MapGeneration.
